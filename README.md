# Finance Dashboard

A responsive finance dashboard built with React and Vite. The app includes a demo login flow, role-aware access, summary cards, charts, transaction management, filters, and light/dark theme support.

## Features

- Demo authentication screen
- Role-based dashboard behavior for admin and viewer modes
- Income, expense, and balance summaries
- Transaction table with filtering
- Charts and insight widgets
- Responsive layout for desktop and mobile
- Light and dark themes

## Tech Stack

- React 19
- Vite
- Recharts
- Lucide React

## Getting Started

1. Install dependencies:

```bash

```

2. Start the development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```
